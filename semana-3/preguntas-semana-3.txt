Pregunta 1 — Selección múltiple
¿Qué es Git?

B.Un sistema de control de versiones para registrar
cambios en proyectos

Pregunta 2 — Selección múltiple
¿Cuál es la función principal de un repositorio?

B.Registrar y organizar los archivos y cambios de
un proyecto.

Pregunta 3 — Selección múltiple
¿Qué hace el comando git add .?

B.Agrega archivos al área de preparación para
registrar cambios

Pregunta 4 — Selección múltiple
¿Qué hace el comando git commit?

A.Guarda un registro de cambios realizados en el
proyecto.
